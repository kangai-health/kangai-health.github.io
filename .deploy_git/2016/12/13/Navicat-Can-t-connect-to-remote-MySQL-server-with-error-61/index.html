Navicat Can't connect to remote MySQL server with error 61

12月 13, 2016 发布在 apache

Navicat无法连接远程数据库问题

标签： apache Navicat

1、Check if your mysql server is listening on a socket with netstat：

  • netstat -tulpen

2、修改/etc/mysql/mysql.conf.d/mysqld.cnf中

3、数据库重启：

  • sudo /etc/init.d/mysql restart
